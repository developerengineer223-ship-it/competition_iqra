# Allama Iqbal Digital Platform

This is a monorepo for the Allama Iqbal digital platform, featuring:
- **api/**: Node.js Express REST API
- **web/**: Next.js minimalist web frontend
- **mobile/**: Flutter app with Nastaleeq rendering
- **db/**: PostgreSQL schema
- **docs/**: OpenAPI spec and documentation

## Quick Start

### API
```
cd api
npm install
npm start
```

### Web
```
cd web
npm install
npm run dev
```

### Mobile
- Open `mobile/main.dart` in your Flutter IDE
- Run on your device/emulator

### Database
- Use `db/schema.sql` to initialize your PostgreSQL database
